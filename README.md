# 📊 AtliQ Mart Supply Chain Analytics | Power BI + SQL Project

## 🔍 Project Overview

AtliQ Mart is facing challenges in maintaining service levels, leading to customer dissatisfaction and potential contract losses.

This project analyzes key supply chain metrics and compares actual performance with targets to identify problem areas.

---

## 🎯 Business Objective

To monitor and improve service level performance using key metrics:

* On-Time Delivery %
* In-Full Delivery %
* OTIF %

---

## 📌 Key Metrics Explained

* **On-Time %** → Orders delivered on or before the agreed delivery date

* **In-Full %** → Orders delivered with full requested quantity

* **OTIF %** → Orders delivered both On-Time and In-Full

* **Line Fill Rate (LIFR)** → % of order lines fulfilled

* **Volume Fill Rate (VOFR)** → % of total quantity delivered

## 📈 Key Insights

* OTIF % is consistently below target across major customers
* Some cities show frequent delivery delays
* Volume Fill Rate is higher than Line Fill Rate → partial fulfillment issue
* Few customers contribute heavily to service failures

---

## 💡 Business Impact

This dashboard helps:

* Track service level performance
* Identify weak areas in supply chain
* Improve customer satisfaction
* Reduce contract loss risk
